# MarketPulse iOS App

Native SwiftUI iOS app that shares core logic and views from `shared/MarketPulseShared`.

## Requirements

- Xcode 15+
- iOS 16.0+ deployment target

## Run

1) Open `MarketPulseApp.xcodeproj` in Xcode.
2) Select a simulator or device.
3) Product -> Run.

The app fetches data from Stooq + FRED (network-only) and supports pull-to-refresh.
